# CSV Pipeline Practice

This repo is my hands-on practice for learning data engineering foundations with Python and pandas.

The goal is to build simple data pipelines that:

- read CSV files
- inspect and validate data
- apply transformations
- write transformed data back to CSV

## Projects in this repo

### 1. Basic Profit Pipeline
File: `src/pipeline.py`

Input:
- `data/jobs.csv`

Transformations:
- calculate profit from sales and cost

Output:
- transformed CSV with profit column

### 2. Advanced Job Pipeline Practice
File: `src/pipeline_practice_2.py`

Input:
- `data/jobs_practice_2.csv`

Planned transformations:
- total cost
- net revenue
- profit
- profit margin percent
- cost per unit
- on-time shipment flag
- late days
- rework flag

Output:
- cleaned and enriched CSV with calculated business columns
